# API Security Dashboard

## Overview
Angular-based dashboard simulating API traffic monitoring and threat detection.

## Features
- JWT-based authentication
- Route protection using Guards
- HTTP Interceptors
- Threat visualization
- Rate-limit monitoring

## Architecture
- Core / Shared / Feature-based structure
- Lazy-ready modular design

## Security Considerations
- Token handling
- Route access control
- Centralized error handling

## Tech Stack
- Angular
- RxJS
- Angular Material
- Chart.js
